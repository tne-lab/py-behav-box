# BehavioralChamber

WIP python program to manage a rat behavioral box with customizable GUI and protocol format.

## Installation

### Dependencies
python3+ (untested but probably 3.5+)
opencv
whisker
zmq

## Usage

```
python BEH_GUI.py 
```
